***********  Это 64-битная версия PyScripter. *************
Для запуска необходимо использовать только 64-битную версию Windows. Обратите внимание, 
что PyScripter будет работать только с установленным Python также в 64-битной версии.

Перед использованием PyScripter, вы должны убедиться, что на вашей машине установлен
Python версии 3.8 или выше. Если это не так, можно скачать нужную версию Python по адресу 
http://www.python.org/ и установить его.

Посетите нашу вики по адресу https://github.com/pyscripter/pyscripter/wiki, чтобы получить
информацию об удалённой отладке и иных темах, связанных с PyScripter.

Новые версии PyScripter доступны на сайте
https://sourceforge.net/projects/pyscripter/files/.

Спасибо за ваш интерес к PyScripter.
